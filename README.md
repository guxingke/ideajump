# Ideajump
like autojump in cli.

find project and use ide open it quickly.

# PRE
- maven 3.3+
- jdk 1.8+
- graalvm 1.0+
- macos // need plutil, json to plist
- yq // yml to json
- /usr/local/bin/idea //diea
- /usr/local/bin/charm // pycharm
- /usr/local/bin/code // vscode

# Build
```bash
./build.sh
```

# Install
```bash
open dist/ideajump.alfredworkflow
```
